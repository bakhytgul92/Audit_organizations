# Audit_organizations
This repository contains an organizational dataset and the complete Python code that processes it.  

You need to do the following steps to run Python files.  

1. Create and activate a virtual environment (optional but recommended)
python -m venv venv

2. Activate in Windows  

venv\Scripts\activate  

or activate in Linux  

source venv/bin/activate  

3. Install all required libraries using pip:

pip install numpy pandas nltk shap matplotlib seaborn xgboost tensorflow scikit-learn imbalanced-learn catboost scipy tensorflow-addons

4. Optional: install Jupyter (if running notebooks)

pip install jupyter notebook

5. Run the Python Script

python main.py
